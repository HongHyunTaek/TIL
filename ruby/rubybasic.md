### 1.puts vs print

```ruby
3.times { print "hello"} # => hellohellohello
3.times { puts "hello"} # => hello
					  # => hello
					  # => hello

```

puts는 개행문자 포함 



### 2. p vs puts

```ruby
string = "hello"
puts string # hello => nil
p string => #"hello" => "hello"
```
